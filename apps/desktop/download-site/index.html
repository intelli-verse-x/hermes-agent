After you install

  1. Open the app and sign in with your work email — you'll get a 6-digit code by email. The session lasts 12 hours.
  2. Everything else is automatic. On sign-in the app fetches your credentials from the portal (gateway token, personal LiteLLM key, VPN profile, Cognito secret), connects the company VPN (expect an admin-password / UAC prompt) and initializes the local Hermes agent. No values to paste.
  3. One prerequisite: install the VPN backend — macOS brew install wireguard-tools, Windows "WireGuard for Windows".
  4. Prefer manual control? Settings → Connect still accepts your own values — anything you set there is never overwritten.

For admins — how to give a user access (step by step)

  1. Open team management: go to admin.intelli-verse-x.ai/admin/team and sign in with your email OTP. You need super admin or owner privilege to see the invite form.
  2. Click "Invite someone" and fill in:
    • Subject — the user's exact work email (or a whole domain to cover a team).
    • Apps — pick the app chips they need, or "All apps".
    • Privilege — owner / manager / viewer.
    • Bundles (advanced) — revenue, growth, support, content, analytics, engineering.
    You can only grant apps, tiers and bundles at or below your own level.
  3. Submit. The access grant is active immediately for the user's next sign-in.
  4. Send the invite yourself. The page shows a copyable invite note — nothing is emailed automatically, so paste it to the user along with a link to this download page.
  5. Done — no credentials to hand out. On the user's first sign-in the app auto-provisions everything server-side: gateway token, a personal LiteLLM key, a personal VPN profile and the Cognito secret.

To revoke access later, delete the user's grant on the same team page — it takes effect at their next login.

First launch on macOS: this build isn't notarized yet, so Gatekeeper will say “IX Agency” is damaged and can’t be opened. It isn't damaged — that's the quarantine flag on unsigned downloads. Drag the app to Applications, then run this once in Terminal and open it normally:
the one-time command xattr -cr "/Applications/IX Agency.app"
On Windows you may see a SmartScreen prompt (choose More info → Run anyway). Only ever install IX Agency from this page.
